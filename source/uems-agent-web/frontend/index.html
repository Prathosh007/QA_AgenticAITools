rocket_launch Getting Started

1

Open the Explorer

Click Open Explorer — no installs, no setup. Requires GitHub Copilot with an active license. Works in any browser.

2

Ask about the codebase

Start typing your question. The Explorer understands the full codebase and can help with:

  • Trace function calls and data flow across repos
  • Understand update, install, and communication workflows
  • Debug log sequences and error patterns
  • Compare platform-specific implementations (macOS vs Linux vs Windows)
3

Search and explore

The Explorer searches across all 38 repositories automatically. It can:

  • Search code patterns across multiple repos simultaneously
  • List classes, structs, protocols, and interfaces
  • Map dependencies between repos and layers
  • Read and explain source files in context
1

Install the extension

Download the VSIX and install it (getting started guide):

  • Terminal: code --install-extension uems-agent-chat.vsix
  • Or VS Code: Extensions (⇧⌘X) → ··· → Install from VSIX…
  • Requires GitHub Copilot with an active license
2

Open Copilot Chat and select an agent

Open Copilot Chat and select UEMS Agent from the participant dropdown. Then choose an agent for your task:

  • Orchestrator — end-to-end SDLC: plan → architect → develop → review → test
  • Explorer — read-only codebase investigation and architecture analysis
  • Delta Reviewer — diff-based code review between branches
  • Document Generator — create and update AI-navigable repo documentation
  • QA — generate structured manual test cases from code changes
3

Describe your task

Tell the agent what you need in plain language. The Orchestrator automatically delegates to specialized sub-agents:

  • Planner analyses requirements and cross-repo dependencies
  • Architect designs the solution with checklist compliance
  • Developer generates production-ready code
  • Reviewer verifies correctness, security, and standards

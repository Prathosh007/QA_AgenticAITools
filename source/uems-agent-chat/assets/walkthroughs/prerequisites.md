## Prerequisites

UEMS Agent Chat requires two things to work:

| Prerequisite | Why |
|---|---|
| **Git** | Clones and syncs agent definitions, searches across repos |
| **GitHub Copilot Chat** | Provides the AI chat interface that powers all agents |

Click **Check Prerequisites** above to verify both are available.

If Git is missing, install it from [git-scm.com](https://git-scm.com) or via your package manager.

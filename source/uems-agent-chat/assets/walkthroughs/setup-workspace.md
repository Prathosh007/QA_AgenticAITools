## Setup Workspace

Clone the UEMS agent repositories to your machine so the agents can search code, trace dependencies, and generate accurate results.

Pick your platform above — this will clone all repos for that platform into a folder you choose.

| Platform | Repos | Languages |
|---|---|---|
| **macOS** | 13 repos | Swift, Objective-C |
| **Linux** | 1 monorepo | Go |
| **Windows** | 18 repos | C, C++, C# |

6 cross-platform repos are included automatically with any platform.

> **Note:** This requires network access to the Zoho Corp Git server.
